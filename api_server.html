/user/find post

post

Поиск пользователя (или создание нового) по запрошенному идентификатору устройства. Возвращается информация о профиле пользователя, лента пользователя и информации о категория/подкатегориях.

post /user/find

Поиск пользователя (или создание нового) по запрошенному идентификатору устройства. Возвращается информация о профиле пользователя, лента пользователя и информации о категория/подкатегориях.

  • Request
  • Response

Body

Type: application/json

Example:

{
  //Идентификатор устройства: string, required
  "id_device": "BF4585465460D",
  //Тип операционной системы: string, required
  "os": "anroid",
  //Версия приложения: string, required
  "version_app": "version_app",
  //Токен: string, required
  "token": "GFDFgfd00D",
  //Ключ: string, required
  "key": "12565D565".
  //Идентификатор города: number, required
  "id_city": 1
}

Успешное выполнение запроса.

Type: application/json

Example:

{

      //Разрешено ли получить уведомления: true - разрешено, false - нет: bool.
      "push": true,
      //Разрешена ли вибрация: true - разрешена, false - нет: bool.
